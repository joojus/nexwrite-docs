## Overview

After purchasing NexWrite, you'll receive a ZIP archive containing all the necessary files to set up your AI-powered platform. This guide will walk you through the contents of your download and explain the purpose of each component.

## Unpacking Your NexWrite Files

Follow these simple steps to access your NexWrite files:

1. **Download the ZIP archive**: After your purchase, you'll receive a link to download the NexWrite package.
2. **Locate the download**: Find the ZIP file in your computer's download folder or the location you specified.
3. **Extract the contents**: Use your computer's built-in unzip tool or a third-party application like 7-Zip to extract the files.

<Tip>
  If you're unsure how to extract ZIP files on your operating system, a quick web search for "how to unzip files on [Your OS]" will provide step-by-step instructions.
</Tip>

## Understanding the File Structure

Once you've extracted the ZIP archive, you'll find the following components:

### 1. documentation.html

This HTML file contains a direct link to the comprehensive online documentation you're currently reading. It's a quick way to access the most up-to-date information about NexWrite.

<Info>
  Pro tip: Bookmark the documentation link for easy access in the future!
</Info>

### 2. support.html

If you encounter any issues during installation or have questions about using NexWrite, this HTML is your go-to resource. It provides information on how to get support from our team.

<Note>
  We're here to help! Don't hesitate to reach out if you need assistance at any point in your NexWrite journey.
</Note>

### 3. nexwrite-server-files.zip

This is the heart of your NexWrite package. This ZIP file contains all the core files required to set up and run NexWrite on your server. Here's what you need to know about it:

- It includes the main application files, configuration templates, and any necessary scripts.
- You'll need to extract and upload these files to your server during the installation process.
- Detailed instructions for handling these files can be found in our [Installation Guide](/get-started/installation).

<Warning>
  Always keep a backup of the `nexwrite-server-files.zip` in a safe place. You may need it for future updates or if you decide to move your installation to a different server.
</Warning>
